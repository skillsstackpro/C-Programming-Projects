# 🐍 Project : 02 Console-Based Snake Game (C Programming)

A classic Snake Game developed in C language. This project steps up from basic logic building to handling **real-time user inputs** and **dynamic coordinates** in a console environment.

## 🚀 Features
- **Real-Time Movement:** Uses `_kbhit()` and `_getch()` for seamless keyboard input (W, A, S, D).
- **Flicker-Free Graphics:** Implemented `gotoxy()` using `<windows.h>` to prevent screen tearing and provide a smooth gaming experience.
- **Dynamic Tail Logic:** The snake grows longer and score increases each time it eats the fruit (F).
- **Wall Collision:** The snake loops through the walls (enters one side, exits the other).

## 🛠️ Tech Stack
- **Language:** C
- **Concepts Used:** Arrays (for tail tracking), Game Loops, Coordinate Geometry (X, Y axes), and Windows API for console cursor manipulation.

## 🎮 How to Play
1. Compile and run the `snake_game.c` file in any C compiler on Windows.
2. Press **W (Up), A (Left), S (Down), D (Right)** to start moving.
3. Press **X** to exit the game.

# Tic-Tac-Toe Game (C Programming) - Semester 01 Project

A classic, console-based Tic-Tac-Toe game developed during my **1st Semester at SZABIST**. This project focuses on implementing core programming logic, array manipulation, and game state management using the C language.

## 🚀 Features
- **Two-Player Mode:** Play as Player X and Player O.
- **Winning Logic:** Automatically detects wins across rows, columns, and diagonals.
- **Draw Detection:** Identifies when the board is full with no winner.
- **Input Validation:** Prevents players from choosing already occupied spots or invalid positions.
- **Clean Interface:** A well-structured console display for a better user experience.

## 🛠️ Tech Stack
- **Language:** C
- **Concepts Used:** 2D Arrays, Functions, Loops (Do-While), and Conditional Logic (If-Else).
